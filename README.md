## 🧠 Problem Statement

95% of Indians do not have a proper financial plan.  
Most people struggle with saving, investing, tax optimization, and long-term financial planning.

---

## 💡 Solution

AI Financial Copilot acts as a **smart financial advisor** that:

- Analyzes income, expenses, and savings  
- Detects financial risks  
- Predicts future outcomes  
- Provides actionable recommendations  

---

## 🔥 Features

- 📊 Smart Dashboard (net worth, savings rate, risk level)  
- 🔥 FIRE Planner (retirement planning)  
- 💯 Money Health Score  
- 🧾 Tax Optimization (Old vs New regime)  
- 📈 Mutual Fund Portfolio Analysis  
- 💑 Couple Finance Planner  
- 🎯 Goal Tracking  
- 🤖 AI Insights (Problem → Impact → Fix)  
- ✏️ Editable Financial Data (dynamic updates)

---

## ⚙️ Tech Stack

- Frontend: HTML, CSS, JavaScript  
- UI Builder: Lovable AI (customized UI)  
- Backend/Auth: Supabase  
- Deployment: Vercel
